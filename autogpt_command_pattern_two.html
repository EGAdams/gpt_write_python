Our Plan

Our system is like a Lego project, where each piece has a specific role and they all fit together to create something amazing.

Object Responsibilities Analogy
Command A Command is like a Lego piece. It has a specific shape (method) and a label (name and description) telling us what it does. It's like a Lego brick with a specific shape and label. You can use it to build your Lego project (program).
CommandRegistry The CommandRegistry is like the Lego baseplate. It holds all the Lego pieces (Commands) and knows where each one is located. It's like a Lego baseplate where you can attach or remove Lego bricks (Commands), or ask a specific brick (Command) to do its job.
command decorator The command decorator is like a Lego machine that turns a normal object into a Lego piece (Command) that can be used in our Lego project (program). It's like a machine that can turn a normal object into a Lego brick that can be used in your Lego project.
