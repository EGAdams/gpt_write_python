Command Center Explanation

This page explains our Python command center using the analogy of a LEGO project.

Command

In our LEGO project, a Command is like a specific LEGO piece. Each piece has:

  • Name: The type of the LEGO piece.
  • Description: What the LEGO piece does in our project.
  • Method: The actual way we use the LEGO piece in our project.

CommandRegistry

The CommandRegistry is like the LEGO project blueprint. It keeps track of all the LEGO pieces (commands) that are available. You can:

  • Register new commands: Add new LEGO pieces to the blueprint.
  • Unregister commands: Remove LEGO pieces from the blueprint.
  • Call a command: Use a specific LEGO piece in the project.

command decorator

The command decorator is like a machine that can turn a normal LEGO piece into a special piece that the blueprint can use. You just take a normal LEGO piece, put it in the machine, and it automatically gets turned into a special piece that the blueprint can recognize and use.

Responsibilities

Object Responsibility Analogy
Command Perform a specific task or action. A specific LEGO piece.
CommandRegistry Manage all the commands. The LEGO project blueprint.
command decorator Turn a normal function into a command. A machine that turns a normal LEGO piece into a special piece.
