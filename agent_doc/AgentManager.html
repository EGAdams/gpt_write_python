AgentManager Class

The AgentManager class is used for managing GPT agents.

Attributes:

next_key:
The next key to be assigned to a new agent.
agents:
A dictionary of agents, where each key-value pair represents an agent and its associated information.
cfg:
The configuration object.

Methods:

__init__:

Initializes the AgentManager class. This method doesn't require any parameters.

__init__(self)

create_agent:

Creates a new agent and returns its key. This method requires three parameters: task, prompt, and model. The task is the task to perform, the prompt is the prompt to use, and the model is the model to use. The method returns the key of the new agent.

create_agent(self, task: str, prompt: str, model: str) -> tuple[int, str]

message_agent:

Sends a message to an agent and returns its response. This method requires two parameters: key and message. The key is the key of the agent to message, and the message is the message to send to the agent. The method returns the agent's response.

message_agent(self, key: str | int, message: str) -> str

list_agents:

Returns a list of all agents. This method doesn't require any parameters. The method returns a list of tuples of the form (key, task).

list_agents(self) -> list[tuple[str | int, str]]

delete_agent:

Deletes an agent from the agent manager. This method requires one parameter: key. The key is the key of the agent to delete. The method returns True if successful, False otherwise.

delete_agent(self, key: str | int) -> bool

Usage with Agent Class:

The AgentManager class can be used in conjunction with the Agent class to manage multiple instances of the Agent class. Here's a simple example:

# Create an instance of the AgentManager class
agent_manager = AgentManager()
# Create an agent
key, response = agent_manager.create_agent(task='task', prompt='prompt', model='model')
# Send a message to the agent
response = agent_manager.message_agent(key=key, message='message')
